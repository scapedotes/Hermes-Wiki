# Fuzzy Matching Engine — 8-Strategy Chain

## Design Principles

When an Agent modifies a file, it needs to locate the text to be replaced. Text generated by an LLM might have subtle differences from the original (e.g., spaces, indentation, escape sequences). Hermes implements an **8-strategy chain** that gradually degrades from exact matching to fuzzy matching, maximizing the success rate of matches.

Inspired by OpenCode's fuzzy matching implementation.

## 8-Strategy Chain

```python
strategies = [
    ("exact", _strategy_exact),                    # 1. Exact Matching
    ("line_trimmed", _strategy_line_trimmed),      # 2. Line Trimmed
    ("whitespace_normalized", _strategy_whitespace_normalized),  # 3. Whitespace Normalized
    ("indentation_flexible", _strategy_indentation_flexible),    # 4. Indentation Flexible
    ("escape_normalized", _strategy_escape_normalized),          # 5. Escape Normalized
    ("trimmed_boundary", _strategy_trimmed_boundary),            # 6. Trimmed Boundary
    ("block_anchor", _strategy_block_anchor),      # 7. Block Anchoring
    ("context_aware", _strategy_context_aware),    # 8. Context-Aware
]

for strategy_name, strategy_fn in strategies:
    matches = strategy_fn(content, old_string)
    if matches:
        # Match found → Execute replacement
        if len(matches) > 1 and not replace_all:
            return content, 0, f"Found {len(matches)} matches, please provide more context"
        new_content = _apply_replacements(content, matches, new_string)
        return new_content, len(matches), None

# All strategies failed
return content, 0, "No match found"
```

## Strategy Details

### Strategy 1: Exact Matching

```python
def _strategy_exact(content, pattern):
    """Direct string matching"""
    matches = []
    start = 0
    while True:
        pos = content.find(pattern, start)
        if pos == -1:
            break
        matches.append((pos, pos + len(pattern)))
        start = pos + 1
    return matches
```

**Applicable Scenarios:** LLM-generated text is an exact match to the original.

### Strategy 2: Line Trimmed


### Strategy 3: Whitespace Normalized

```python
def _strategy_whitespace_normalized(content, pattern):
    """Collapses multiple spaces/tabs into a single space"""
    def normalize(s):
        return re.sub(r'[ \t]+', ' ', s)
    
    pattern_normalized = normalize(pattern)
    content_normalized = normalize(content)
```

**Applicable Scenarios:** LLM-generated text has inconsistent whitespace quantities.

### Strategy 4: Indentation Flexible

```python
def _strategy_indentation_flexible(content, pattern):
```

**Applicable Scenarios:** LLM-generated text has different indentation levels.

### Strategy 5: Escape Normalized

```python
def _strategy_escape_normalized(content, pattern):
    """Converts escape sequences to actual characters"""
    def unescape(s):
        return s.replace('\\n', '\n').replace('\\t', '\t').replace('\\r', '\r')
    
    pattern_unescaped = unescape(pattern)
    if pattern_unescaped == pattern:
        return []  # No escape sequences, skip
    
    return _strategy_exact(content, pattern_unescaped)
```

**Applicable Scenarios:** LLM-generated text contains literal escape sequences.

### Strategy 6: Trimmed Boundary

```python
def _strategy_trimmed_boundary(content, pattern):
```

### Strategy 7: Block Anchoring

```python
def _strategy_block_anchor(content, pattern):
    """Anchors based on first and last lines, using similarity matching for the middle part"""
    # Unicode Normalization
    norm_pattern = _unicode_normalize(pattern)
    norm_content = _unicode_normalize(content)
    
    pattern_lines = norm_pattern.split('\n')
    first_line = pattern_lines[0].strip()
    last_line = pattern_lines[-1].strip()
    
    # Find positions of first and last line matches
    for i in range(len(norm_content_lines) - pattern_line_count + 1):
        if (norm_content_lines[i].strip() == first_line and 
            norm_content_lines[i + pattern_line_count - 1].strip() == last_line):
            
            # Calculate similarity for the middle part
            content_middle = '\n'.join(norm_content_lines[i+1:i+pattern_line_count-1])
```

### Strategy 8: Context-Aware

```python
def _strategy_context_aware(content, pattern):
    """Line-by-line similarity matching, with a 50% threshold"""
    pattern_lines = pattern.split('\n')
    content_lines = content.split('\n')
    
    for i in range(len(content_lines) - pattern_line_count + 1):
        block_lines = content_lines[i:i + pattern_line_count]
        
        # Calculate line-by-line similarity
        high_similarity_count = 0
        for p_line, c_line in zip(pattern_lines, block_lines):
            sim = SequenceMatcher(None, p_line.strip(), c_line.strip()).ratio()
            if sim >= 0.80:  # 80% similarity for a single line
                high_similarity_count += 1
        
        # At least 50% of lines must have high similarity
        if high_similarity_count >= len(pattern_lines) * 0.5:
            matches.append(...)
```

**Applicable Scenarios:** Overall content has more than 50% line similarity.

## Unicode Normalization

```python
UNICODE_MAP = {
    "\u201c": '"', "\u201d": '"',  # Smart double quotes
    "\u2018": "'", "\u2019": "'",  # Smart single quotes
    "\u2014": "--", "\u2013": "-", # Dashes
    "\u2026": "...", "\u00a0": " ", # Ellipsis and non-breaking space
}

def _unicode_normalize(text: str) -> str:
    """Normalizes Unicode characters to standard ASCII equivalents"""
    for char, repl in UNICODE_MAP.items():
        text = text.replace(char, repl)
    return text
```

## Superiority Analysis

### Matching Success Rate

| Scenario | Exact Matching | 8-Strategy Chain |
|-------------------------------------|----------------|------------------|
| Fully Identical                     | ✅             | ✅               |
| Leading/Trailing Whitespace Differences | ❌             | ✅ Strategy 2/6  |
| Indentation Differences             | ❌             | ✅ Strategy 4    |
| Escape Sequence Differences         | ❌             | ✅ Strategy 5    |
| Smart Quotes                        | ❌             | ✅ Strategy 7    |
| Subtle Middle Content Adjustments   | ❌             | ✅ Strategy 7/8  |
